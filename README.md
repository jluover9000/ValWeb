# ValWeb 

Interactive Valentine's Day proposal website with Bubu & Dudu theme.

## Features
- Playful "No" button that escapes on hover
- "Yes" button with progressive effects (grows, glows, sparkles)
- Celebration page with fireworks and animations

## Setup
1. Clone repo
2. Add video files to `img/` folder
3. Open `index.html` in browser

## Deploy to GitHub Pages
1. Push to GitHub
2. Settings → Pages → Deploy from `main` branch
